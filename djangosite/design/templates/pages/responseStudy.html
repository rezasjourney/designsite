Create a graphic design

Instructions:

Your task for this HIT is to use an automatic layout tool to create a graphic design, and report on the automatic updates. The tool will provide automatic updates when you move or resize elements. You must like and dislike at least 3 different updates(for both) while you are using the interface.

The layout you are creating should contain all the elements, and they should all be visible. You create a layout that clearly conveys the information of the design and looks good. If elements overlap, the text should still be readable.

Evaluation:

Your layout will be evaluated by other MTurk workers. The layout does not need to be perfect but reasonable effort is required. Your actions and time spent creating the layouts are recorded. If the layouts are obviously made very quickly will little effort, or the layouts are judged to be very similar to each other, the HIT will be rejected.

You must provide some explanation for disliked updates.

How to Use the Suggestion Interface:


This interface is designed to provide you with suggestions when you are creating a layout. As you move elements, the interface will make suggestions and automatically adjust the elements. If you don't want elements to move, you can fix them to prevent automatic updates (see below).

If the interface is working, the loading icon is shown: . You can still move elements while the interface is working.

How to Use the Interface:

  1. Change How Elements Automatically Move
    If you want to fix elements so they won't move automatically, you can select one or more elements and press the red lock icon at the top , or the lock icon beside the element. The yellow lock icon means the elements will stay near that current location and only move a bit. The open green lock means the elements can move anywhere in the layout.
  2. Undo/Redo Automatic Suggestions
    If you want to go back to a previous layout, you can press the Undo button at the top. You can also use the Redo button to go forward.
    3. {% endif %}
  3. Resizing Elements
    To resize elements, drag the round circles on the corners.
  4. Changing the Number of Lines
    To change the number of lines for each element, drag the red diamonds left or right.
  5. Saving a Layout
    When you find a layout that you want to save, press the 'Save Layout.' Saving a layout will automatically reset the main layout to a random initial layout. You can remove the layout later by pressing the remove button beside it. If you mouse over the saved layout, it will appear as the main layout. Click the saved layout to set this as the main layout.
